# hiband-h5页面模板

## 开发环境

模板用的数据是`config/data.js`里面的`data.dev`

```
yarn dev
```

## 打包测试站

模板用的数据是`config/data.js`里面的`data.test`

```
yarn build:test
```

## 打包正式站

模板用的数据是`config/data.js`里面的`data.prod`

```
yarn build:prod
```

## 启动静态资源服务

**mode**: 分别为dev|test|prod，分别对应开发环境|测试站|正式站

```
yarn server:[mode]
```

## 发布到服务器

> TODO: 上传接口有问题

```
yarn publish:[mode]
```
